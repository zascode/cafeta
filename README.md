# CAFETA static site (GitHub Pages)

This repository contains a static website version of cafeta.de without a web shop.

Open points you should customize before publishing:
- Replace the Amazon link in `index.html` and `cafeta/index.html`.
- Review legal pages (`datenschutz`, `agb`, `shipping-returns`) for correctness.
- Optionally replace the newsletter mailto flow with a dedicated newsletter provider.

Deployment (GitHub Pages):
1) Push this repository to GitHub.
2) In the repository settings, enable GitHub Pages for the default branch and root folder.
3) Configure the custom domain (cafeta.de) and ensure the DNS records point to GitHub Pages.
